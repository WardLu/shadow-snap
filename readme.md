# 图片字幕生成器 (Subtitle Image Generator)

这是一个专门通过“背景切割”技术生成电影质感多行字幕图片的网页工具。它可以将你输入的每一行文字，配合图片底部的背景片段，完美融合成一张长图。

## ✨ 核心特性
1.  **切割感背景**：每一行字幕条的背景都精准提取自原图底部，确保视觉上的连贯性与“电影感”。
2.  **多行输入**：在文本框内直接回车换行，即可增加新的字幕条。
3.  **水印系统 (V2.0)**：支持自定义文字水印，可自由调节位置、透明度、大小及颜色。
4.  **极速交互**：
    - **点击即上传**：直接点击图片预览区域即可选择新图片。
    - **一键删除**：支持清除当前图片，恢复到初始待机状态。
5.  **全球化字体**：支持中文、英文、泰文、越南文、阿拉伯文，内置多款免费商用字体。
6.  **实时预览**：所有参数调整（包括水印）均为毫秒级即时反馈。
7.  **离线安全**：纯浏览器前端实现，图片不会上传到服务器。

## 🚀 如何使用
1.  双击打开目录下的 `index.html` 文件（建议使用 Chrome, Edge 或 Safari 浏览器）。
2.  点击 **“选择图片”** 按钮或直接将图片**拖入**左侧区域。
3.  在右侧文本框输入内容。**按下 Enter 键换行**。
4.  根据需要调整：
    - **字幕条高度**：影响行与行之间的间距。
    - **字体大小**：调整文字显示。
    - **安全边距**：控制文字离左右边缘的距离。
5.  满意后，点击 **“保存并下载图片”** 即可。

## 📂 文件说明
- `index.html`：网页主程序。
- `style.css`：界面样式（深色高级感）。
- `script.js`：核心绘图引擎逻辑。
